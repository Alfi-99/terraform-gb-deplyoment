Arsitektur High Availability Tanpa Biaya
Lab ini mengimplementasikan strategi Blue/Green Deployment yang biasanya mahal, namun dioptimalkan untuk AWS Free Tier. Setiap interaksi sistem dicatat ke Amazon DynamoDB (log terpusat) dan dikelola dengan Terraform. Hapus semua resource setelah selesai praktikum untuk menghindari tagihan di luar batas waktu/kuota gratis.
⚠ Perhatian: Meskipun Elastic Beanstalk gratis, instance EC2 di dalamnya menggunakan kuota 750 jam/bulan. Gunakan instance t2.micro agar tetap gratis.

Instruksi Praktik
Lab Steps — Blue/Green Koperasi Merah Putih
Ikuti 10 langkah strategis untuk membangun infrastruktur Koperasi Merah Putih secara berurutan.
STEP 01
FREE ✓
🌐
VPC Networking
VPC · Subnet · IGW
Bangun pondasi jaringan aman multi-AZ
  • Buat VPC: 10.0.0.0/16 region ap-southeast-1
  • Buat Public Subnet untuk Load Balancer
  • Buat Private Subnet untuk EB dan RDS
  • Konfigurasi Internet Gateway dan NAT Gateway (simulasi via NAT Instance)
STEP 02
FREE ✓
🛡️
Security Groups
IAM · Security Layers
Pasang sistem keamanan berlapis
  • ALB SG: port 80/443 dari internet
  • EB SG: port 80/8080 hanya dari ALB
  • RDS SG: port 3306 hanya dari EB dan Lambda
  • Konfigurasi IAM Role untuk akses S3 dan DynamoDB
STEP 03
FREE ✓
🪣
Amazon S3 Storage
Versioning · Lifecycle
Siapkan gudang penyimpanan artefak & media
  • Bucket 1: Deployment Artifacts (zip hasil build)
  • Bucket 2: Application Media (user uploads)
  • Aktifkan SSE-S3 encryption dan Versioning
STEP 04
FREE ✓
📂
Amazon EFS
Standard Mode · Multi-AZ
Setup shared storage antar instance EB
  • Buat EFS File System mode General Purpose
  • Configure Mount Targets di setiap private subnet
  • Setup Access Points untuk direktori aplikasi
STEP 05
FREE ✓
☁️
Elastic Beanstalk
t2.micro · Node.js 20
Deploy Environment BLUE & GREEN
  • Deploy Environment BLUE (versi stabil)
  • Deploy Environment GREEN (versi baru)
  • Konfigurasi ALB Weighted Listener Rules
STEP 06
FREE ✓
🚪
API Gateway
REST API · Regional
Setup pintu masuk request API
  • Buat REST API dengan Regional endpoint
  • Configure Stage 'prod' dan Access Logging
  • Setup Usage Plan dan Throttling (1000 RPS)
STEP 07
FREE ✓
λ
AWS Lambda
Node.js · VPC Access
Implementasi eksekutor serverless
  • Lambda POST Handler: Write to RDS & DynamoDB
  • Lambda GET Handler: Read from RDS
  • Configure VPC Subnets dan SG untuk Lambda
STEP 08
FREE ✓
🗄️
Amazon RDS
MySQL 8 · db.t3.micro
Database relasional utama
  • Launch RDS db.t3.micro (20 GB storage)
  • Simpan kredensial di Secrets Manager
  • Aktifkan Automated Backups
STEP 09
FREE ✓
📊
Amazon DynamoDB
Logging · TTL
Implementasi buku log terpusat
  • Buat tabel Log dengan Partition Key: LogID
  • Configure TTL (7 hari) untuk auto-deletion
  • Setup GSI untuk pencarian berdasarkan LogLevel
STEP 10
FREE ✓
🚀
CI/CD Pipeline
Actions · Amplify
Otomatisasi deployment & frontend
  • Setup GitHub Actions workflow (test & build)
  • Configure Amplify untuk hosting frontend
  • Test Rollback via Terraform variable change
